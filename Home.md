# Projeto Portfólio Pessoal — Wiki

Bem-vindo(a) à documentação do projeto.

Esta wiki contém:
- Casos de Teste — repositório de casos de teste organizados por ID e funcionalidade.
- Regras de Negócio — documento das regras de domínio que orientam o comportamento do sistema.
- Modelos (templates) para criar novos casos de teste e regras de negócio.

Como usar
1. Procure pela seção desejada na barra lateral.
2. Para adicionar um novo Caso de Teste: copie o template `TC-Template` e preencha um novo arquivo seguindo a nomenclatura `TC-XXX-<titulo-compacto>.md`.
3. Para adicionar ou alterar uma Regra de Negócio: copie o template `BR-Template` e salve com `BR-XXX-<titulo-compacto>.md`.
4. Sempre referencie issues/PRs relacionadas e adicione evidências (links para imagens ou anexos).
